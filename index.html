HTML

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • When applying a value like margin we can use a short cut by saying 0 20 0 20: this will result in margin on both left and righ also can be written like 0 20px.
  • When linking a css style sheet you use link tag in html and type "rel="decriptive name of file" href="location of file here" "

Git

  • git checkout -b "Branch-name": checks you into a new branch and you can name the branch
  • git status: shows what changes you have in your current branch and see if you can add
  • git add -a: adds all the files to your status and then later commit
  • git commit -m "Your informative notes on changes": commits all the changes in your status to your current branch
  • git pull origin main: Good practice to see if our local branch is sync with the bas branch on github before creating a pull request
  • git push origin "current branch name": pushes our changes to the remote branch on github
  • In github you can see the push to the online branch and then creat a pull request to merge it to the mainbranch
